Common HTML tags

Tag Name Description
<html> HTML Defines the root of an HTML document
<header> Header Defines a header for a document or section
<h1> to <h6> Headings Used to define headings of different sizes
<footer> Footer Defines a footer for a document or section
<section> Section Defines a section in a document
<p> Paragraph Defines a paragragh within an HTML document
<img> Image Used to embed an image in a HTML page
<video> video Used to embed a video in an HTML page
<div> Division Container that divides the page into sectors
